Core styles are what you need to write your custom css. They include custom properties, custom media queries and utilities.

## Media queries

Inside the core style, we provide a set of breakpoints which you can use while building components and applications. Here the complete list of the available custom media queries you can use:

### Viewport

```css
@media (--from-extra-small);
/* min-width: 30em => 480px */

@media (--from-small);
/* (min-width: 48em) => 768px */

@media (--from-medium);
/* (min-width: 60em) => 960px */

@media (--from-large);
/* (min-width: 80em) => 1280px */

@media (--from-extra-large);
/* (min-width: 100em) => 1600px */
```

### Orientation

```css
@media (--portrait);
/* (orientation: portrait) */

@media (--landscape);
/* (orientation: portrait) */
```

### System preferences

```css
@media (--dark-scheme);
/* (prefers-color-scheme: dark) */

@media (--light-scheme);
/* (prefers-color-scheme: light) */

@media (--allow-motion);
/* (prefers-reduced-motion: no-preference) */
```

## Elevations

Our shadows are defined based on the Elevations schema. Each level makes elements cast a specific type of shadow.

export const CardStyles = {
  inlineSize: 130,
  blockSize: 130,
  backgroundColor: "var(--dimmed-2)",
  borderRadius: 8,
  display: "flex",
  placeContent: "center",
  placeItems: "center",
};

<Stack direction="row" columnGap={24}>

    Z1
