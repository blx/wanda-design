The separator component is a simple horizontal rule which defines the end of a content section and the start of a new one.

## Usage

The component doens't have any property and you can just drop it inside your code.

<LiveArea fill>
  <Separator />
</LiveArea>

```jsx
<Separator />
```

<Snackbar type="info">
  The separator renders a <b>hr</b> element. If you are just adding visual
  separation and not a content-related separation you must add{" "}
  <b>aria-hidden</b> attribute.
</Snackbar>
