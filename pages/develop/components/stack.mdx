The `Stack` is a utility component that helps with the box modeling and layouts. Based on flexbox it allows content to be spaced and positioned along a single axis.

## Usage

If you need to place or add space between elements this component comes with handy props that make your life easier. Check this example:

<LiveArea fill>
  <Stack columnGap={8} direction="row" fill={false}>
    <Select>
      <option value="1">Option 1</option>
      <option value="2">Option 2</option>
      <option value="3">Option 3</option>
    </Select>
    <Button>Confirm</Button>
  </Stack>
</LiveArea>

```jsx
<Stack columnGap={8} direction="row" fill={false}>
  <Select>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>

  </Select>

```

## Inline

The Stack component by default is rendered as `flex` element, so it will fill all the available space. If you want to disable this behaviour you can use the `inline` boolean prop to make it as `inline-flex` and prevent the grow:

<LiveArea fill horizontalAlign="unset">
  <Stack inline columnGap={8} direction="row" fill={false}>
    <Select>
      <option value="1">Option 1</option>
      <option value="2">Option 2</option>
      <option value="3">Option 3</option>
    </Select>
    <Button>Confirm</Button>
  </Stack>
</LiveArea>

<Snackbar kind="info" title="Remember">
  If you nest multiple Stack components make sure to properly set the alignment
  because flex properties are inherited by children.
</Snackbar>
