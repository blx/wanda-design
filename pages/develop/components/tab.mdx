## Anatomy

```jsx
import { Tab } from "@wonderflow/react-components";

export default () => {
  return (
    <Tab.Root>
      <Tab.Panel label="Tab item 1">Tab panel 1</Tab.Panel>
      <Tab.Panel label="Tab item 2">Tab panel 2</Tab.Panel>
    </Tab.Root>
  );
};
```

## Accessibility

Adheres to the [Tabs WAI-ARIA design pattern](https://www.w3.org/TR/wai-aria-practices-1.2/#tabpanel).
