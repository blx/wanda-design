## Accessibility

Adheres to the [`alert` role requirements](https://www.w3.org/TR/wai-aria-1.2/#alert).

The `Snackbar` component uses the `role="alert"` by default with inplicits `aria-live="assertive"` and `aria-atomic="true"` attributes. Add those to change the default values.
