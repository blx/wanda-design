With the `Title` component you can add title using the style from the Wanda typography. You can change the level of the title and the `line-height`.

## Usage

By default the Title component just renders a `span` element with the visual style of `h1`.

<LiveArea direction="column" horizontalAlign="start">
  <Title as="h1" level="1">
    Semantic H1
  </Title>
  <Title as="h2" level="2">
    Semantic H2
  </Title>
  <Title as="h3" level="3">
    Semantic H3
  </Title>
  <Title as="h4" level="4">
    Semantic H4
  </Title>
  <Title as="h5" level="5">
    Semantic H5
  </Title>
  <Title as="h6" level="6">
    Semantic H6
  </Title>
</LiveArea>

## Style and semantic

You can notice how the visual appearance of the title is decoupled from its html tag, this way you can keep the headings structure while using different styles. For example you can render an `h2` element with the `as` property, while using the style for an `h5` by defining the `level` property to `5`.

<Snackbar type="warning" title="About semantic">
  It's important to use the correct tag and in order to keep the content
  accessible to screen readers and assistive technologies in general.
</Snackbar>
